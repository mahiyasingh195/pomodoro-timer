# Pomodoro Timer

A responsive Pomodoro timer built using HTML, CSS and JavaScript to help users stay productive using the Pomodoro Technique.

## Features

- 🍅 Focus and Break modes
- ▶️ Start, Pause and Reset timer
- ⏱️ Live countdown timer
- 📈 Tracks completed Pomodoro sessions
- ✨ Interactive UI with visual feedback
- 📱 Responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript
